## 초기 설정

---
>  setup 스크립트는 최초 한번 실행 해야 합니다. -> 네트워크 생성


## 사용법

---

### 실행 스크립트 명령어

```
./mysql -> docker compose를 통해 한번에 실행합니다.
```

### Container 폴더 설명
* 종료 명령 스크립트 
  * 각 **_continer 의 `stop` 스크립트 실행 
* 백그라운드 실행 명령 스크립트
  * 각 **_continer 의 `start` 스크립트 실행
